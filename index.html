3 veces más que un viajero solo

No es solo que paguen más por noche — es que ocupan la misma habitación y se quedan las mismas noches que una pareja, pero el hotel cobra una tarifa sensiblemente mayor.

Ingreso total estimado por estancia (€)

Insight clave

Una familia ocupa la misma habitación que una pareja, se queda las mismas 3,7 noches, pero genera 561 € frente a 361 €. El hotel cobra más por el mismo recurso, sin coste adicional. La tarifa media diaria sube de 99 €/noche a 151 €/noche sólo cambiando el tipo de cliente — no la habitación.

Hotel Bookings Dataset · Portugal 2015–2017 · 119.390 reservas

Esta visualización demuestra que los datos del sector hotelero contienen más información de la que habitualmente se analiza. La narrativa scrollytelling permite guiar al lector de forma progresiva, generando tensión antes de revelar cada insight. La principal limitación del dataset es que no incluye el coste real de las comisiones ni datos de satisfacción del cliente — con esa información la historia sería aún más precisa. El uso del color de forma semántica (verde = rentable/fiel, rojo = riesgo/coste) y la animación progresiva buscan que cada dato impacte en el momento correcto.
Esta práctica utilizó Claude (Anthropic) como herramienta de apoyo en dos fases: (1) Generación del código — el HTML, CSS y JS fueron generados a partir de prompts detallados que especificaban la estructura narrativa, los datos exactos de cada acto y los requisitos técnicos de animación e interactividad. (2) Verificación — una vez generado el código, se comprobó que todos los datos mostrados concordaban con el dataset original (hotel_bookings.csv) y los análisis propios. Todo el proceso fue supervisado detalladamente, revisando y ajustando el output en cada paso.
